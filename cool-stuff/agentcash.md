## What Is It?

**AgentCash** gives AI agents a wallet and native **x402 payments** — one balance with access to every API on the internet. It's the payment/wallet infrastructure layer for autonomous agents, enabling agents to pay for APIs and services on-chain without human intervention.

Partners with **SuedeAI** to form a combined stack:
- **AgentCash** → wallet + x402 native payments (agents that pay)
- **SuedeAI** → musical/generative ability, identity, provenance, governance (agents that prove)

Result: **One balance. One trust layer.**

---

## Key Concepts

- **x402 native**: Built around the x402 payment protocol (HTTP 402 — payment required)
- **Agent wallet**: Agents get their own balance, not relying on human-managed keys for every tx
- **Universal API access**: One balance covers any API on the internet that accepts x402 payments
- **Pairs with SuedeAI**: Identity + provenance layer from SuedeAI + payment layer from AgentCash = full agent economic stack
